Required hardware:
   o STM32 or MBHP_CORE_STM32

===============================================================================

This application provides a tiny console for sending commands to the application
via MIDI interface

Use $(MIOS32_PATH)/tools/mios_terminal to open the MIDI connection, to
receive messages and to send commands to the console.
